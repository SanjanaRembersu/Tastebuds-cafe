# 🍴 Tastebuds Cafe  

Welcome to **Tastebuds Cafe**, a modern restaurant website built with **HTML, CSS, and JavaScript**.  
This is part of my web design portfolio to showcase my skills in creating beautiful, responsive, and client-ready websites.  

---

## 🌟 Features  
- Elegant landing page with a hero banner  
- Navigation bar with **Home, About Us, Menu, Contact**  
- Menu section with images and prices  
- Order Now button for easy call-to-action  
- Responsive design (works on desktop & mobile)
